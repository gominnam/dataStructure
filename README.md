## 학습으로 얻는 것
```
1. 자신의 생각을 코드로 구현하는 기술의 익숙함
2. 라이브러리를 알고 쓰기 위함
3. 효율적인(최적의) 문제 해결력
4. 기본기를 배움으로 오는 자신감
5. 개인적인 즐거움
```

------------------------------------------------------

+ 학습 순서
  + LinkedList
    + append(추가), delete(삭제), retrieve(출력)
    + removeDups(중복제거), KthToLast(끝에서 k번째 값)
  + Stack
    + push(추가), pop(top 삭제, top 반환), peek(top 반환), isEmpty(top Null 체크)
